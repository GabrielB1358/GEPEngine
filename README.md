# Games Engine Programming Unit Project
Homemade Game Engine capable of:
- Rendering Models & Textures into a 3d scene
- Altering & playing SFX / Audio Tracks from custom locations
- Polling & handling user Input (default example uses it to move a 'player' entity)
- Handles both Keyboard, Mouse **and** Controller input
- Allows users to create basic games with ease

# Controls for default example
**A** & **D**  or  for Left/Right
**Space** for Jump

# Specifics
- Written in Visual Studio using C++
- OpenGL used for all graphical rendering and visuals
- OpenAL used to handle all audio usage
